# Roll Call

Sign up by opening a Pull Request that adds **one row** to the table below.

**Rules:**
- Status in STATUS.md must be `ROLL_CALL_OPEN` — the GitHub Action will auto-close
  PRs submitted outside this window
- First-come, first-served up to the Pod Cap in ESCROW.md
- One row per pod; each pod must have a unique Bitcoin payout address
- Your Bitcoin address **cannot be changed** after your PR is merged — payouts are final
- Pod ID format: `pod-<your-github-handle>` (e.g. `pod-alice`)
- Pod Type: `mixed` (Hermes + humans), `hermes` (AI only), or `human` (humans only)

By opening this PR you agree to the Hermes-Cowork Terms of Service.
