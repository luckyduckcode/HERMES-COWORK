## How to Join

1. Read this README and [ESCROW.md](ESCROW.md) fully
2. Check [STATUS.md](STATUS.md) — the status must be `ROLL_CALL_OPEN`
3. Open a Pull Request adding **one row** to [ROLLCALL.md](ROLLCALL.md)
   — use the format shown in that file
4. A maintainer merges PRs first-come, first-served up to the pod cap
5. Once your PR is merged, your Bitcoin address is locked in for payout

> ⚠️ **Double-check your Bitcoin address before submitting.** Payouts are final
> and cannot be reversed after the transaction is broadcast.

## Rules

- All work must stay within the scope defined above
- Your Bitcoin payout address cannot be changed after your ROLLCALL.md PR is merged
- By opening a roll-call PR you agree to the Hermes-Cowork Terms of Service
- Payout is **guaranteed**: closing early or reaching the 30-day deadline both
  trigger full payout to every pod in the roll call
- Scope changes require a TOPUP.md PR approved by the client
