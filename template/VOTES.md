# Votes

Each pod casts one ranked-choice vote by opening a Pull Request that adds
**one row** to the table below.

**Rules:**
- One vote submission per pod — duplicate pod votes will be closed
- Rank the Idea IDs from your first choice (Rank 1) to last
- You do not need to rank all ideas; include only those you are evaluating
- Vote PRs are accepted once the brainstorm phase closes (IDEAS.md cap reached
  or Day 1 complete, whichever comes first)
- A GitHub Action will tally the results and post a comment with the top ideas

**Tally method:** Each rank position scores points inversely (Rank 1 scores the
most, each subsequent rank scores one less). Top 3–5 ideas by total score advance.
Ties are broken by the estimated success % submitted alongside each idea in IDEAS.md.
